# Thymeleaf 3 with Servlet 3 Example
Thymeleaf 3 using the servlet 3 specification without web.xml

## Prerequisite
- Maven 3.3.3+
- Java 8+
 
## Running the sample
Run with `mvn tomcat7:run`. Visit site on [http://localhost:8080/thymeleaf-servlet3-example/](http://localhost:8080/thymeleaf-servlet3-example/).
